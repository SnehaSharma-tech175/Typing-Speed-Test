# Typing Speed Test Using Python Tkinter

## Introduction

The **Typing Speed Test** is a GUI-based application developed using **Python** and the **Tkinter** library. It allows users to test their typing speed by typing a given sentence. The application calculates the typing speed in **Words Per Minute (WPM)** and displays the result.

## Features

* Simple and user-friendly graphical interface.
* Displays a predefined sentence for typing.
* Starts the timer automatically when the user begins typing.
* Calculates typing speed in Words Per Minute (WPM).
* Displays the typing speed instantly after completion.
* Easy to modify with different typing passages.

## Technologies Used

* Python 3

## How to Run the Project

1. Install Python 3 on your computer.
2. Save the project file as `TypingSpeedTest.py`.
3. Open a terminal or command prompt.
4. Navigate to the project folder.
5. Run the following command:

```bash
python TypingSpeedTest.py
```

## How It Works

1. The application displays a sentence that the user must type.
2. The timer starts automatically when the user clicks inside the text box.
3. After typing the sentence, the user clicks the **Done** button.
4. The application calculates the typing speed in **Words Per Minute (WPM)**.
5. The typing speed is displayed on the screen.
